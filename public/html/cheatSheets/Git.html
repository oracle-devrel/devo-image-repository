Create
Action Command
Create an empty Git repository git init
Create an empty Git repository in the specified directory git int < directory>
Create a local working copy of a remote repository git clone
Clone a remote repository (no authentication) git clone git://host/path/to/repo.git
Clone a remote using SSH(requires credentials on the hosting server) git clone ssh://< user>@host/path/to/repo.git
Clone to specific folder git clone < remoteRepoURL> < directory>
Clone a single branch from a remote repository git clone [-b | - -branch] < branchName> - -singlebranch < remoteRepoURL>

Action Command

Remove local (untracked) files from the working tree

git clean

Dry run: shows you what files would be deleted if you ran git clean

git clean -n

Remove all untracked files (ignores directories)

git clean -f

Remove a specific untracked file

git clean -f <filePath>

Remove untracked directories as well as files

git clean -fd

Remove any ignored, untracked files/directories

git clean -xf[d]

Interactive mode

git clean -i[d]

git rebase

Abort a rebase

git rebase - -abort

git reflog

git reset

git reset

Reset HEAD to the indicated commit

git reset <commitID>

Saving changes
Action Command

Add files to the index

git add

Add all new, modified, and deleted files to a commit

git add [ . | -A | --all ]

Add modified and deleted files

git add -u

Add 1 or more files to the staging area

git add <file1> <file2> … <fileN>

Selectively add parts of a file to a commit

git add [-p | --patch]

Don’t add the file(s), just show if they exist and/or will be ignored

git add [-n | --dry-run] [options] <files>

Sends you into an interactive utility where you can perform all the various subcommands in one location

>git add - -interactive

 

Selectively add parts of a file for a commit

>git add [-p | - -patch]

Advanced (add: excluding files)

Add all files, but exclude one

>git add --all -- :!path/to/file/dontcheckmein.txt

Add all files, but exclude several

git add --all -- :!path/to/file1 :!path/to/file2 :!path/to/folder1/*

Store changes in a temporary location, providing you with a clean working directory

git stash

Store all your untracked and ignored files

git stash

Stash file and annotate with title

git stash push -m “<title>” <file>

Stash untracked changes

git stash [-u|- -include-untracked]

Stash untracked and ignored files

git stash [-a| - -all]

Restore your most recent local changes to the staging area without removing them from the tracked stash list

git stash apply

Restore your most recent local changes to the staging area and removes them from the tracked stash list

git stash pop 

 

Empty the stash list by removing all the stashes

git stash clear

View your stash list

git stash list 

Show a summary of diffs between the current stash (i.e., top of the stash stack) and the commit at the time the stash was saved

git stash show

 

Inspect
Action Command

>Show what revision and author last modified each line of a file

git blame

>Ignore whitespace when comparing the parent’s version with the child’s to find where the lines came from

>git blame -w <file>

>Detect moved or copied lines within a file

>git blame -M [<num>] <file>

>Detect lines moved or copied from other files that were modified in the same commit

>git blame -C [<num>] <file>

>Add a new branch to an existing branch, view all existing branches, or delete a specific branch

git branch

>Display branches

>git branch [- -list]

>Show remote branches in addition to local branches.

>git branch [-r | - -remotes]

>Restrict list of displayed branches to those matched by <pattern>

>git branch - -list <pattern>

>Sort branches by day

>git branch - -sort=-commiterdate (sort branches by date)

>See where all your branches are tracking from

>git branch -vv

>List branches merged into main

>git branch - -merged main

>List branches merged into HEAD (i.e., the tip of the current branch)

>git branch - -merged

>List branches that have not been merged

>git branch - -no-merged

>See a list of files with conflicts after a merge

>git diff - -name-only - -diff-filter=U

>Show commit logs

git log

>Limit the number of commits returned by the command

>git log -n <numCommits>

>Return just the commit messages and condense output to one line per commit

>git commit - -oneline

>Include which files were altered and the relative number of lines that were added or deleted

>git log - -stat

>Display the patch representing each commit

>git log -p

>Show commits for the specified author

>git log - -author=”<pattern>”

>Show only commits that contain the specified file

>git log <file>

>Show only those commits that occurred between <since> and <until>

>git log <since>..<until>
